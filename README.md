# ⚡ VaultTecDairy ⚡
REST service for Vault-Tec-Dairy (backend)

### Stack (current): 
  1. Spring Boot 2
  2. Hibernate
  3. Maven
  4. H2 Database
  5. Swagger 2
  6. Liquibase
